1.4 Changing the Fiscal Year

Out of the box, Illiad assumes the fiscal year starts on July 1st. To change this, change the file grails-app/conf/BootStrap.groovy to something like:

class BootStrap {
    def init = { servletContext ->
        metridoc.utils.DateUtils.FY_START_MONTH = Calendar.JULY
    }
    def destroy = {
    }
}

You should also change the start date on the run-illiad script.
